# ETEC - Desenvolvimento de Sistemas 🚀

Este repositório foi criado para organizar e centralizar todas as atividades, projetos e exercícios práticos realizados durante o curso técnico de **Desenvolvimento de Sistemas** na ETEC.

## 📌 Objetivo
O foco principal é o aprendizado de lógica de programação, estruturas de dados e o domínio de tecnologias modernas, mantendo um histórico da minha evolução como desenvolvedor(a).

---

## 📂 Projetos em Destaque

### 1. Calculadora de Consumo de Energia (`/consumo-energia`)
Um programa em Python desenvolvido para calcular o consumo mensal de eletrodomésticos em kWh e o custo estimado em Reais.
* **Tecnologias:** Python 3.x
* **Conceitos aplicados:** Entrada/saída de dados, operações matemáticas, formatação de strings e tipos primitivos (`float`, `str`).

---

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **Linguagens:** Python (inicialmente)
* **Controle de Versão:** Git & GitHub
* **Editor de Código:** Visual Studio Code (VS Code)
* **Terminal:** Git Bash / PowerShell
